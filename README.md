# Hanggi

### 주요 기능
1. 미디어 쿼리를 이용한 반응형 작업
2. 네비게이션 scroll 됐을 때 if문 활용
3. 메인 비주얼은 background-attachment: fixed;
4. button에 hover했을 때 스타일 변경
5. 광고 배너는 가로 슬라이드 구현 (li의 width 만큼 ul이 이동하는 방식)
6. sns 영역에 a링크 연결

- HTML5, CSS3, jQuery

- 채식한끼 반응형 웹 사이트입니다.
